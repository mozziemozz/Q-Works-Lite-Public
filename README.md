## Prerequisites

The following tools need to be installed:

- PowerShell 7
- Azure Function Core Tools
- Azure CLI
- Python 3.11.9
- Node.JS
    - M365 CLI
- PowerShell Modules: 
    - Microsoft.Graph.Applications
    - Microsoft.Graph.Users
    - Az.Accounts
    - Az.Functions
    - Az.KeyVault
    - Az.Storage
    - Az.Resources
    - Az.Websites
